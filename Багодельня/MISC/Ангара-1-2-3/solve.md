# Ангара-1-2-3 (серия задач)

## Overview

**Категория:** MISC  
**Описание:** Серия из трёх связанных задач. Решать задачи можно только по порядку, т.к. следующие части связаны с предыдущими.

**Цель миссии:** Запустить ракету в космос.

**Адрес:** http://31.207.77.216:4016/

### Задачи серии

1. **Ангара-1: Вход в ОС** - Восстановить пароль для входа в систему управления полетами
2. **Ангара-2: Коды запуска** - Найти коды запуска ракеты (Флаг: BugCTF{код_запуска})
3. **Ангара-3: Параметры пуска** - Найти правильные параметры для пуска ракеты

---

## Задача 1: Ангара-1 - Вход в ОС

### Описание

Нужно войти в систему управления полетами BugOS, но пароль потерялся. Авторизацию настраивал приглашенный специалист, который оставил описание по восстановлению пароля.

### Анализ

Заходим на сайт и сразу же видим кнопку "Напомнить пароль", скачиваем текстовый файл.

![ангара 1](../../img/ангара-1.png)

### Решение

Открываем текстовик и предполагаем, что просто раскладку поменяли. Находим сервис для изменения раскладки и копируем текст из текстовика.

![ангара 1-2](../../img/ангара-1-2.png)

Используем полученный пароль и заходим в BugOS.

![ангара 1-3](../../img/ангара-1-3.png)

---

## Задача 2: Ангара-2 - Коды запуска

### Описание

Теперь необходимо найти коды запуска ракеты в системе BugOS.

### Анализ

Скачиваем:
- Картинку из вкладки "Девчонки"
- Документы по запуску (архив)

Архив запаролен. Используем метод LSB для извлечения информации из изображения.

![ангара 1-4](../../img/ангара-1-4.png)

Нам сообщают, с помощью какого словаря можно найти пароль для архива.

### Решение

С помощью john извлекаем хэш и получаем пароль:

![ангара 1-5](../../img/ангара-1-5.png)

Разархивируем архив и получаем ссылку на документ.

![ангара 1-6](../../img/ангара-1-6.png)

Находим код запуска: увеличиваем шрифт и меняем цвет текста.

![ангара 1-7](../../img/ангара-1-7.png)

### Флаг

Код запуска является флагом для Ангара-2: `BugCTF{код_запуска}`

## Задача 3: Ангара-3 - Параметры пуска

### Описание

Есть коды запуска, остается найти нужные параметры для пуска ракеты. В прошлом году, сразу после настройки BugOS, был выполнен пробный пуск с правильными параметрами.

### Анализ

Пытаемся запустить ракету, но не получается.

![ангара 2](../../img/ангара-2.png)

Смотрим ответ от сервера и видим лог URL.

![ангара 2-1](../../img/ангара-2-1.png)

### Решение

Переходим по ссылке, видим ответ сервера. Вспоминаем, что в описании написано: первый запуск был правильный. Ищем первый запуск.

![ангара 3-1](../../img/ангара-3-1.png)

Предполагаем, что это MD5. Проверяем и точно!

![ангара 3-2](../../img/ангара-3-2.png)

Кодируем цифру 1 и получаем хэш.

![ангара 3-3](../../img/ангара-3-3.png)

Заменяем наш хэш на только что полученный и получаем правильный запуск.

![ангара 3-4](../../img/ангара-3-4.png)

### Флаг

Вводим данные в систему запуска и получаем флаг.
